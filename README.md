# Lista de Tarefas

Este é um projeto de uma lista de tarefas simples desenvolvido em React. Ele permite adicionar, marcar como concluídas e deletar tarefas. As tarefas são armazenadas no `localStorage` do navegador para que persistam entre as sessões.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Deletar tarefas individuais
- Deletar todas as tarefas
- Persistência de dados usando `localStorage`

## Tecnologias Utilizadas

- React
- CSS
